# mit-python-study

John Guttag의 『Introduction to Computation and Programming Using Python』(MIT Press, 3rd ed.)를 정독하며 푼 연습문제 풀이 저장소입니다. 회사 일과 별개로 CS 기초를 다시 세우는 것이 목적입니다.

## 진행 방식

- 하루에 한 챕터씩 읽습니다.
- 각 차시의 교재 finger exercise를 직접 풀어 코드로 남깁니다.
- 이해도 점검용으로 스스로 낸 문제 3개도 차시마다 함께 풉니다.

## 저장소 구성

```
chNN-solve.py   # N차시(챕터 N) 문제 풀이 코드
```

- 파일명은 챕터 번호 두 자리(`ch01`, `ch02` …)를 따릅니다.
- 코드 안 주석으로 어떤 문제인지 표시하고, 그 아래에 풀이를 둡니다.
